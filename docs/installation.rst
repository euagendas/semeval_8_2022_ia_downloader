============
Installation
============


Stable release
--------------

To install semeval 2021 task 8 downloader, run this command in your terminal:

.. code-block:: console

    $ pip install semeval_8_2021_ia_downloader

This is the preferred method to install semeval 2021 task 8 downloader, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for semeval 2021 task 8 downloader can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/hide-ous/semeval_8_2021_ia_downloader

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/hide-ous/semeval_8_2021_ia_downloader/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/hide-ous/semeval_8_2021_ia_downloader
.. _tarball: https://github.com/hide-ous/semeval_8_2021_ia_downloader/tarball/master
